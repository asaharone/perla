🚨 Рекомендація #1: Створити llms.txt

Файл /llms.txt повинен містити: опис бренду англійською, ключові продуктові категорії, науковий підхід, сертифікації (HACCP, ISO 9001:2015, ISO 22000:2005), і preferred citation format. Це instant win для AI-видимості.

🚨 Рекомендація #2: Author Profiles для E-E-A-T

Для YMYL-контенту (здоров'я/добавки) критично мати: іменовані автори блогу з credentials, сторінка "Наукова рада" з фото + кваліфікацією, Person schema з jobTitle, alumniOf, sameAs. На сайті є 7 експертів на /about-us, але вони не пов'язані з конкретним контентом.

ТОП-10 сильних сторін

1. Виняткова довіра бренду

4.9★ рейтинг, 7000+ відгуків, 150K+ клієнтів, лого Vogue/Forbes/Elle — найкращий social proof серед українських supplement брендів. Celebrity endorsements (олімпійська чемпіонка Різатдінова, співачка Астаф'єва) додають aspirational appeal.

2. Award-winning пакування та брендинг

1-е місце Dieline Awards (Health category). Archetype-based product architecture з color-coded системою для 5 жіночих архетипів. Ребрендинг від Orchidea Agency створив holistic identity.

3. Глибокий product content

Продуктові сторінки (напр. Triple Magnesium) мають 14,695+ символів контенту з H2 структурою, детальними інгредієнтами, науковими посиланнями, FAQ, та відгуками. Benefit-driven copy (outcomes > features).

4. Наукова експертиза

7 іменованих експертів на /about-us з академічними credentials (Dr. Заячківська — доктор медичних наук, професор фізіології). Сертифікації HACCP, ISO 9001:2015, ISO 22000:2005.

5. Міжнародна sourcing narrative

Omega-3 з Ісландії, Вітамін D3 зі Швейцарії, Морський колаген з Франції, Вітамін K2 з Норвегії, Сквален з Португалії — premium international sourcing як genuine differentiator.

6. Responsive design system

6 breakpoints (430-1916px), Tailwind CSS design system, Onest + Piazzolla font pairing з повною Cyrillic підтримкою, WebP оптимізація, touch-optimized carousels.

7. Omnichannel дистрибуція

DTC сайт + Rozetka + EVA + Makeup.com.ua + Hotline.ua + KAPSULA. Міжнародна експансія: perlaholistic.com (EN/PL), EU entity в Польщі.

🔴 Негайно (Тиждень 1-2) — Critical Fixes

1. Додати H1 теги на всі сторінки

Замінити CSS-класи .p-title на семантичні <h1> теги. 6 сторінок потребують fix. Найбільший SEO impact.

2. Виправити Product schema

Видалити дублікат Product з name "Star rating snippet aggregator". Залишити один коректний Product з AggregateRating, Offer, Brand.

3. Додати meta description та title для /store

Title: "Вітаміни та добавки | Каталог Perla Helsa — купити з доставкою". Meta description з USP та CTA.

4. Виправити focus visibility (accessibility)

Замінити outline:none на :focus-visible з custom ring (2px offset у brand green). Глобально через Tailwind @layer utilities.

5. Додати skip-navigation link

Перший focusable елемент: <a href="#main" class="sr-only focus:not-sr-only">Перейти до контенту</a>

🟠 Короткострокові (Місяць 1) — High Impact

6. Впровадити email capture та базові email flows

Welcome series, abandoned cart, post-purchase sequence, replenishment reminders. Highest-ROI action для 150K+ клієнтів без CRM.

7. Додати FAQPage schema на продуктові сторінки

Кожна сторінка з FAQ accordion має отримати JSON-LD FAQPage markup. SSR-rendered.

8. Створити llms.txt + додати lastmod в sitemap

llms.txt англійською з brand overview, key products, certifications. Sitemap з lastmod для всіх URLs.

9. Виправити SSR для /ordering

Забезпечити server-side rendering для payment/delivery info. Критично для SEO та accessibility.

10. Додати author profiles до блогу

Іменовані автори з credentials, Person schema з jobTitle, alumniOf. "Рецензовано [Ім'я, Credential]" на всіх health-статтях.

Benefit-driven CTA замість generic navigation link. A/B тест різних варіантів.
